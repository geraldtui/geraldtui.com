Designed, developed, and launched an AI-powered Kubernetes agent used by the platform
teams to troubleshoot complex cluster issues and gain richer observability across their
environments. Built on AWS Bedrock with Claude, the agent reasons over live cluster state
and surfaces root causes that would otherwise take hours of manual investigation.

Highlights:

- Built an autonomous troubleshooting agent with Python and the Strands agent framework.
- Delivered an intuitive Next.js interface for the platform teams to query and act on cluster insights.
- Powered reasoning with Claude on AWS Bedrock to diagnose complex, multi-layered issues.
- Improved observability and cut mean-time-to-resolution for cluster incidents.
