Codified the entire AWS cloud platform as Infrastructure-as-Code using Terraform and
Terragrunt, turning environment provisioning into a fast, repeatable, and auditable
process. Defining the cloud foundation in code made it consistent across regions and
accounts and removed slow, error-prone manual configuration.

Highlights:

- Provisioned AWS cloud infrastructure from code, cutting deployment time from hours to minutes.
- Standardized environments across regions and accounts with reusable Terraform/Terragrunt modules.
- Made the cloud foundation version-controlled, auditable, and easy to reproduce.
